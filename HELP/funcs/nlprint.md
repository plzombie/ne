# Функция nlPrint

## Синтаксис

```c
void nlPrint(const wchar_t *format, ...);
```

## Объявлена в

> "nyan_log_publicapi.h"

> "nyan_publicapi.h"

## Описание

Печатает сообщение в журнал сообщений. Идентична функции *wprintf*.

* **format** - строка форматирования.

## Возвращаемое значение

Нет

## Потокобезопасная

Да
